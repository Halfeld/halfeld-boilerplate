### How to use

```sh
# Clone project
git clone git@github.com:Halfeld/halfeld-boilerplate.git my-awesome-website

# Install dependencies
bower install
npm install

# Run
npm start
# localhost:3000
```

+ Jekyll
+ Stylus
+ Browserify
+ Babel
